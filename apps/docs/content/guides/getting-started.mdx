# Getting Started

Welcome to IndustrialKit! This guide will help you get started with the library.

## Installation

```bash
npm install industrialkit
```

## Basic Setup

Add the plugin to your `tailwind.config.js`:

```js
module.exports = {
  content: [
    // ...
    "node_modules/industrialkit/dist/**/*.{js,mjs}"
  ],
  plugins: [
    require('industrialkit/plugin')
  ]
}
```
