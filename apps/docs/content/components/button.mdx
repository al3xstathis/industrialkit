# Button Component

The Button component is a fundamental building block for user interactions.

## Usage

```tsx
import { Button } from 'industrialkit'

function Example() {
  return (
    <Button variant="primary">
      Click me
    </Button>
  )
}
```

## Variants

- `default`
- `primary`
- `secondary`
- `danger`

## Sizes

- `sm`
- `md`
- `lg`
